# 💰 Smart Expense Tracker

A smart, responsive web application built with pure JavaScript to help users manage their daily expenses, set monthly budgets, and track remaining balances in real-time.

🌐 **Live Demo:** [Click Here to View Project](https://github.io)

---

## 🚀 Key Features

* **Set Personal Budget:** Users can input and update their total budget anytime.
* **Persistent Data (Local Storage):** Transactions and budget data remain saved even after refreshing the page.
* **Real-time Calculations:** Automatically calculates Total Expenses and Available Balance dynamically.
* **Category Filtering:** Filter transactions by Food, Bills, Entertainment, or Other categories instantly.
* **Delete Transactions:** Easily remove incorrect entries with an auto-updating scoreboard.
* **Overspending Alert:** Visual alert (red indicator) when the available balance drops to zero or below.

---

## 🛠️ Tech Stack Used

* **HTML5:** Structured semantic layout.
* **CSS3:** Responsive dashboard cards and custom warning animations.
* **JavaScript (ES6):** Core application logic, including DOM Manipulation, Array Reducers (`.reduce()`), Array Filtering (`.filter()`), and Web Storage API.

---

## 📂 Project Structure

* `index.html` - Main application dashboard interface.
* `style.css` - Custom design and visual indicator layouts.
* `app.js` - Live calculations, Local Storage functions, and action event listeners.

---

## 📝 How to Run Locally

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Start tracking your daily expenses!
